# 🩺 AI-Based Skin Disease Detection System

## 📌 Project Overview
This project is a Machine Learning / Deep Learning based system that predicts skin diseases from input images. It helps in early detection and classification of common skin conditions using Artificial Intelligence.

---

## 🎯 Objective
- To build a model that can accurately classify different types of skin diseases
- To assist in early diagnosis using image-based prediction
- To apply machine learning techniques in healthcare

---

## 🧠 Technologies Used
- Python 🐍
- NumPy
- Pandas
- Matplotlib / Seaborn
- TensorFlow / Keras
- OpenCV
- Scikit-learn

---

## 📂 Dataset
- The dataset contains images of different skin diseases
- Images are categorized into multiple classes
- Data preprocessing and augmentation techniques are applied

---

2. Data Preprocessing

4. Model Building (CNN)
5. Model Training
6. Model Evaluation
7. Prediction
